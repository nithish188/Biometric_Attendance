# 🚀 IoT Biometric Attendance System

An advanced IoT-based biometric attendance system with real-time tracking, automated SMS alerts, and cloud data logging.

---

## 📌 Overview
This system captures student attendance using fingerprint authentication and ensures accurate, real-time recording through NTP-based time synchronization. Attendance data is uploaded to the cloud, and absentee alerts are automatically sent to parents via SMS.

---

## ✨ Key Features
- 🔐 Fingerprint-based secure attendance  
- 📡 Real-time cloud logging using ThingSpeak  
- 📲 Automated SMS alerts for absentees  
- ⏱️ Accurate time synchronization using NTP  
- 💾 EEPROM-based student data storage  
- 👥 Supports up to 75 students  

---

## 🧠 System Architecture
Fingerprint → Attendance Processing → Cloud Upload → SMS Alert

---

## ⚙️ Working Flow
1. Student scans fingerprint  
2. System verifies identity  
3. Attendance is recorded with timestamp  
4. Data is sent to cloud platform  
5. After attendance window, absentees are identified  
6. SMS alerts are sent automatically  

---

## 🛠️ Setup Instructions
1. Upload Arduino code  
2. Upload ESP8266 code  
3. Configure WiFi credentials and API key  
4. Connect all hardware modules  
5. Power ON the system
